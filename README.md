# 合工大教务轮子 Python Ver.

## 使用须知
1. 从命令行形式启动 `main.py` ，初次使用请按照 `requirements.txt` 安装依赖库。
2. 支持 macOS/Windows/Linux
3. 合肥校区暂未测试。

## 功能
* 导出课表为 ics 文件，可导入至手机日历及添加提醒
* 顺便查查成绩
